# SportsBooking

Система бронирования спортивных площадок.

## Функционал

- Регистрация и авторизация пользователей
- JWT аутентификация
- Хеширование паролей BCrypt
- CRUD операции для спортивных площадок
- Бронирование площадок
- Загрузка изображений
- Swagger документация API
- WPF клиентское приложение
- Тесты xUnit

## Используемые технологии

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- JWT
- BCrypt

### Frontend
- WPF

### Тестирование
- xUnit
- EF Core InMemory

## Структура проекта

- SportsBooking.Api — серверное API
- SportsBooking.Wpf — WPF клиент
- SportsBooking.Data — база данных
- SportsBooking.Models — модели
- SportsBooking.Tests — тесты

## Запуск проекта

1. Запустить SportsBooking.Api
2. Запустить SportsBooking.Wpf
3. Использовать Swagger для тестирования API

## Автор

Студенческий проект по дисциплине разработки клиент-серверных приложений.
